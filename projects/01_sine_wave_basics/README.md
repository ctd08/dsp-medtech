# Project 01 - Sine Wave Basics

## Goal
Build intuition for how frequency, amplitude, phase, and sampling rate change a sine wave in the time domain.

## Concepts trained
- sampling frequency
- time vector
- sine generation
- parameter variation

## Tasks
- generate a 5 Hz sine wave
- generate a 10 Hz sine wave
- compare two amplitudes
- compare two phases
- change the sampling rate and observe the plot

## Questions to answer
- what changes when frequency increases?
- what stays the same when only phase changes?
- what gets worse when the sampling rate is too low?
